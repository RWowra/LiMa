Activating and Returning Machine-Based License of Type FlexNet Embedded (FNE)

To activate or return a machine-based license with FlexNet Embedded, you need an activation ID.

  • To activate a machine-based license online
  • To activate a machine-based license offline
  • To return a machine-based license

To activate a machine-based license online

If your PC is connected to the internet, you can activate the machine-based license online. The License Manager will contact the ETAS license server located in the internet and download the license directly on your PC.

To activate the license online, do the following:

  1. In the Device menu, select Online Activation.

  2. Enter the activation ID printed on the ETAS Entitlement Certificate and click Activate.

  3. The license is activated.

To activate a machine-based license offline

If your PC is not connected to the internet, you need to activate your activation ID offline. This involves files creation containing request information that needs to be uploaded to the ETAS license portal, and transferring so called response data to your PC. To perform the activation, you can use a USB stick, an e-mail, a network share etc.

To activate the license offline, do the following:

  1. On the PC where the software is installed

    1. Select in the Device menu of the License Manager the entry Offline Activation.

    2. Type the activation ID printed on the ETAS Entitlement Certificate in the field Activation ID.

    3. At Step 1 click Create Request.

      A BIN file is created. The default file name ends with _activaterequest.bin.

    4. Save the BIN file.

  2. On a PC with internet connection:

    1. Login to the ETAS License and Download Portal (http://www.etas.com/support/licensing) with your activation ID.

    2. In the Devices menu, select Upload Capability Request.

    3. Click Browse and select the file ending on _activaterequest.bin.

    4. Click Upload to process the file.

      A request BIN file is created.

    5. Download the new file and save it.

  3. On the PC where the software is installed:

    1. In the Device menu of the License Manager, select the entry Offline Activation.

    2. At Step 2 click Complete Activation and upload the BIN file that you have downloaded from the ETAS License and Download Portal.

    3. Click Upload.

  4. The license is activated.

To return a machine-based license

Note  

It is only possible to return a machine-based license for FlexNet Embedded in the offline modus.

  1. On the PC where the software is installed

    1. Select in the Device menu of the License Manager the entry Offline Return.

    2. Select an activation ID from the drop-down menu.

    or

    1. In the table of the License Manager right-click the activation ID shown in the column Serial Number.

    2. Select Copy Serial Number to Clipboard.

    3. Select in the Device menu the entry Offline Return.

    4. Paste the activation ID in the field Activation ID.

  2. At Step 1

    1. Click Ask for Return.

      A BIN file is created. The default file name ends with _returnrequest.bin.

    2. Save the BIN file.

    3. On a PC with internet connection, login to the ETAS License and Download Portal (http://www.etas.com/support/licensing) with your activation ID.

    4. In the Devices menu, select Upload Capability Request.

    5. Click Browse and select the file ending on _returnrequest.bin.

    6. Click Upload to process the file.

      A response BIN file is created.

    7. Download the new file and save it.

    8. In the Device menu of the License Manager select the entry Offline Return.

  3. On the PC where the software is installed:

    1. At Step 2 click Delete License and upload the BIN file that you have downloaded from the ETAS License and Download Portal.

      A window appears.

    2. Click OK.

  4. At Step 3

    1. Click Confirm Return.

      A confirm BIN file is created which ends with returnconfirm.bin.

    2. Save the BIN file.

    3. On a PC with internet connection, login to the ETAS License and Download Portal (http://www.etas.com/support/licensing) with your activation ID.

    4. In the Devices menu select Upload Capability Request.

    5. Upload the confirm BIN file.

    6. Click Upload.

      The license portal asks you to download a new BIN file. You can ignore this request.

      The return process is completed.

      Note  

      Only if you perform the last step, the license has been returned.
