Setting User Options

Tools

In the Tools menu, you can perform the following actions:

Reset Warning Dialogs

If you have ticked the Never show dialog again checkbox for a warning message, this warning message is no longer displayed. By clicking Reset Warning Dialogs, you can enable all warnings again.

Tools options

When you open the Options dialog, you can make the following settings:

Checkout retries

This option defines how often the system shall try to borrow a license. After the last retry, the result of this retry is shown.

Skip on host down

If the host cannot be accessed, with this option the checkout of the license can automatically be skipped.

Refresh status information at startup

The status information is automatically refreshed when starting the ETAS License Manager.

Remember window layout and positions

The current size and position of the ETAS License Manager window is stored.

Offline mode

The ETAS License Manager changes into "offline mode". Now, only machine-based and borrowed licenses can be used. Due to internal processes, it is possible that in some cases when using the offline mode you will get a notification if you do not have write permission.

Autoborrow visible

Shows three additional columns in the license manager: "Autoborrow Active", "Autoborrow Interval" and "Autoborrow Renewal Period".

Operating version visible

Displays the version of the license.

Remaining days visible

Shows the remaining usage days available in the Grace Mode.

Re-parse descriptions

Shows the license description files of the installed software and add-ons.

Policy type visible

Shows the license type used (Publisher, Embedded, No License, Grace Mode).

Serial number visible

Shows the activation ID for the machine-based license.

Device settings

In the Device menu, if you open the Settings dialog, you can perform the following actions:

Copy to Clipboard

Copies the Host Identifier to the clipboard.

Test Connection

Checks if your PC can access the FlexNet Operations server.

Copy Address to Clipboard

Copies the Internet Proxy address to the clipboard.

Copy User ID to Clipboard

Copies the user ID to the clipboard.

Proxy Password

Allows to enter a password for the Internet Proxy.

Proxy Mode

Allows you to choose between three different proxy modes:

  • None: no internet proxy is used
  • Manual: you can choose the proxy address, the user ID and the proxy password
  • Auto: the License Manager searches automatically for the proxy address, the user ID and the proxy password
Open Requests in Explorer

Opens a new File Explorer window containing the request files.

Open Responses in Explorer

Opens a new File Explorer window containing the response files.

Time-out

Allows you to configure the duration of the connection to ETAS FlexNet Operations (ETAS FNO).
