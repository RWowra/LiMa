Borrowing and Returning a License

The borrowing mechanism allows you to work offline (i.e. without being connected to the license server) even if you are using a server-based license.

Note  

Borrowing a license is only possible if you are using a server-based license. For more information about a server-based license, see License Types.

  • To borrow a license

  • To return a borrowed license

To borrow a license

  1. If you want to borrow specific licenses, select them in the table.

  2. In the License menu, select one of the following options:

    • Borrow Selected Licenses

      or

    • Borrow All Licenses
  3. In the calendar view, select the date until which you would like to borrow the licenses.

  4. Click OK.

  5. The source information changes into Borrow. The borrow expiry date is displayed. You can now use the corresponding ETAS software offline until the borrow expiry date is reached.

If you would like to continue to use the ETAS software longer than determined, you can borrow it again. If you have finished using the ETAS software earlier than planned, you can return it earlier to the license server.

To return a borrowed license

If you have finished using the ETAS software earlier than planned, you can return it earlier to the license server.

Note  

A borrowed license can only be returned by the person who borrowed it. It cannot be returned by somebody else.
  1. Make sure that the ETAS software whose license you would like to return is not running.
  2. If you want to return specific licenses, select them in the table.

  3. In the License menu, select one of the following options:

    • Return Selected Borrowed Licenses

      or

    • Return All Borrowed Licenses

  4. The source information changes into Server.
