Upgrading Licenses

You can use the online and offline upgrade feature to upgrade your licenses to a higher version for your licensed ETAS products. When using this feature, the ETAS License Manager checks the ETAS License Portal (FNO server) for the latest state of your licensed ETAS products.

Note  

You can only upgrade your licenses if you are using a valid machine-based license. You can not upgrade your licenses if you are using floating licenses, user-named licenses, or working via a remote desktop session (RPD) .

If you can not upgrade your licenses, contact your ETAS Tool Coordinator.

To upgrade licenses online

If your computer is connected to the Internet, you can upgrade the licenses online.

  1. In the Device menu, select Online Upgrade.

    The License Manager lists machine-based licenses for both FlexNet Embedded and FlexNet Publisher technology.

  2. Select the licenses that you want to upgrade.

  3. Click Upgrade Selected Licenses to upgrade the licenses.

To upgrade licenses offline

If your computer is not connected to the Internet, you must create files with the request information that must be uploaded to the ETAS License Portal for review and transfer the response data to your computer.

STEP 1 STEP 2 STEP 3
Create request file on your computer. Request response file on computer online. Copy response file to your computer.

 

  1. On the computer where the software is installed, select in the Device menu of the License Manager the entry Offline Upgrade.

    The License Manager shows you machine-based licenses for both FlexNet Embedded and FlexNet Publisher technology.

  2. Click Check if online upgrade is possible. If it is possible, use the To upgrade licenses online otherwise continue with next steps.

  3. Select the Activation IDs that you want to upgrade.

  4. At Step 1

    1. Click Create Request.

      A request file for the upgrade is created.

    2. Save the request file on a device and transfer it on the PC used in step 2.

  5. On a computer with internet connection:

    1. In the Device menu, select the entry Offline Upgrade.

    2. At Step 2, click Upload Request.

      The ETAS License Manager will upload the request file that contains the upgrade information.

      You get a response file.

    3. Save the response file and transfer it to the PC used in step 3.

  6. On the PC where the software is installed:

    1. In the Device menu of the License Manager, select the entry Offline Upgrade.

    2. At Step 3, click Process Response and upload the response file that you have received.

      The licenses are upgraded.
