Migrating FlexNet Publisher Licenses to FlexNet Embedded

Note  

The migration of a FlexNet Publisher license to FlexNet Embedded is only possible with a valid maintenance contract.
  • To migrate machine-based licenses online
  • To migrate machine-based licenses offline

To migrate machine-based licenses online

If your PC is connected to the internet, you can migrate the machine-based licenses online.

  1. In the Device menu, select Online Migration.

  2. Select the licenses that you want to migrate.

  3. Click Migrate Selected Licenses to migrate the licenses to FlexNet Embedded.

To migrate machine-based licenses offline

If your PC is not connected to the internet, you need to create files containing request information that needs to be uploaded to the ETAS license portal and transferring so called response data to your PC.

Do the following:

  1. On the PC where the software is installed, select in the Device menu of the License Manager the entry Offline Migration.

  2. Select the activation IDs that you want to migrate.

  3. At Step 1

    1. Click Create Request.

      A request file for the migration is created.

    2. Save the request file on a device and transfer it on the PC used in step 2.

  4. On a PC with internet connection:

    1. In the Device menu, select the entry Offline Migration.

    2. At Step 2, click Upload Request. The ETAS License Manager will upload the request file that contains the migration information.

      You get a response file.

    3. Save the response file and transfer it to the PC used in step 3.

  5. On the PC where the software is installed:

    1. In the Device menu of the License Manager, select the entry Offline Migration.

    2. At Step 3, click Process Response and upload the response file that you have received.

      The licenses are migrated to FlexNet Embedded.
