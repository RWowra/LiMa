Checking the License Status

The ETAS License Manager contains one entry for each installed feature.

To check the license status, do the following:

  1. If you want to get the status of specific features, select them in the table.

  2. In the License menu, select one of the following options:

    • Refresh Selected Licenses

      or

    • Refresh All Licenses

  3. The check-out of the licenses is started.

The following information is displayed:

State (icon)

This column displays a graphical symbol indicating the current state of the license. After refreshing, one of the following symbols is shown:

A valid license file is installed for the product.

No valid license file is available for the product.

A valid license file is installed for the product, but it will expire within the next 30 days.
Product

Indicates the main ETAS product the license is valid for (e.g. INCA, ASCET).

Feature

Specifies the content the license is valid for.

For any content that requires a separate license, e.g. an add-on to the main product, you need a separate license.

Status

Indicates the current licensing status for the ETAS software. The following status values are available:

Status

Explanation

Unknown

The information has not yet been refreshed.

In use

A license is installed and in use, i.e. the software is running.

Installed

A license is installed, but not in use, i.e. the software is not running.

Grace Mode

The Grace Mode Licenses can be either permanent or limited for a certain period. If you are borrowing a license from a license server, for instance, this is always limited to a certain date, which you can enter upon borrowing. The Grace Mode offers a defined contingent of days of usage in which the software can be used without a license. Only the actual days of usage will be counted. The available contingent depends on the ETAS software offering the Grace Mode. It can be disabled or provides up to 14 days of usage. Once this contingent has been used up, the ETAS software can be run only after installing a valid license. is active.

Not available

There is no license installed. It is not possible to use the corresponding ETAS software.

Heartbeat failed

The product requires heartbeat exchanging with the server, but it failed.

Checkout successful

The license is checked-out.

Checkout pending

The checkout has not yet been completed. You can cancel the checkout.

Checkout cancelled

The checkout has been cancelled.

Borrow

The license has been borrowed. If automatic borrow is activated, the system has automatically prolonged the borrowed license.

Borrow pending

The system has tried to automatically prolong the borrowed license but could not complete the action.
Feature Version

Displays the license version requested by the installed software. The software version can differ from the license version.

Operating Version

Displays the version of the license.

Note  

The license version and the product version are not identical. Example: according to the license model, the license version 7.2 can be valid for the product versions 7.0 and 7.1. For details see your license contract.
Policy Type

Indicates which license type is in use. The following license types are available:

Policy Type

Explanation

Publisher

FlexNet Publisher licensing, using license files

Embedded

FlexNet Embedded licensing, using trusted storage

No license

No available license

Grace Mode

Defined contingent of usage days in which the software can be used without a license
Source

Indicates where the license is managed. The following license sources are available:

Source

Explanation

Not available

The information has not yet been refreshed.

Local

The license is referring to the MAC address of your PC and cannot be used on any other PC.

Local licenses allow you to work offline without being connected to a license server.

Server

The license is provided via a license server.

The license file needed for this license type contains the reference to the license server and is installed locally on your PC.

Borrow

The license is checked out from the license server, which allows you to work offline. It will become invalid after a predefined time. If you want to continue to use the ETAS software after the expiry date, you need to borrow the license again.

The license file needed for this license type contains the reference to the license server and is installed locally on your PC.
Expiration Date

Indicates when the license will expire.

Note  

The status information can be automatically refreshed when starting the ETAS License Manager. For more information, see Setting User Options.
Remaining Days

Indicates the remaining usage days available.

Grace Mode Days Left

Indicates when the Grace mode will expire.

Autoborrow Active

Indicates if the autoborrow mode is active.

Autoborrow Interval

Indicates the number of days for which the license is to be borrowed.

Autoborrow Renewal Period

If autoborrow is active, indicates the days remaining before the license borrowing is extended by the number of days given in Autoborrow Interval.

Serial Number

Shows the activation ID for the machine-based license.

Terminal Service

Shows if the PC allows that you can use the FNP license for remote working.
